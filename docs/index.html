Why leaklatch

Built for the TypeScript developer's inner loop — not a heavyweight platform.

⚡

Sub-second pre-commit

Lowest false positives

Layered rules + entropy + placeholder filters mean it stays quiet unless it's real.

Get started in 30 seconds

No global install needed.

# Install the pre-commit hook in your repo
$ npx leaklatch install

# Run a one-off scan any time
$ npx leaklatch scan

# Scan the whole tree in CI
$ npx leaklatch scan --all --json
